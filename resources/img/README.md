# Plugin Icons

This directory contains all the SVG icons for the AI Contract Assistant plugin.

## Icon Files

### Main Plugin Icon
- **icon.svg** (64x64px) - Main plugin icon displayed in the plugin manager

### Feature Icons (32x32px)
- **ask-ai-icon.svg** - Ask AI chat feature
- **summary-icon.svg** - AI Summary generation
- **obligations-icon.svg** - Obligations extraction
- **clauses-icon.svg** - Key clauses extraction
- **playbook-icon.svg** - AI Playbook execution
- **approval-icon.svg** - Approval workflow

## Design Specifications

- **Color Scheme:** Primary blue (#2667ff) matching your design system
- **Style:** Clean, modern, minimalist
- **Format:** SVG (scalable vector graphics)
- **Sizes:** 
  - Main icon: 64x64px
  - Feature icons: 32x32px

## Customization

You can customize these icons by:
1. Editing the SVG files directly
2. Replacing with your own designs
3. Converting to PNG if needed (OnlyOffice supports both)

## Icon Descriptions

- **icon.svg** - Blue rounded square with checkmark, representing AI-powered contract assistance
- **ask-ai-icon.svg** - Chat bubble with AI sparkle, representing conversational AI
- **summary-icon.svg** - Document with text lines and sparkle, representing document summarization
- **obligations-icon.svg** - Checklist with checkmarks, representing task/obligation tracking
- **clauses-icon.svg** - Document sections with numbered circles, representing clause extraction
- **playbook-icon.svg** - Book with AI sparkle, representing AI playbook execution
- **approval-icon.svg** - Checkmark in circle, representing approval workflow

All icons are ready to use and properly referenced in `config.json`.
